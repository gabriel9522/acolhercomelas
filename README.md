## Acolher com Elas

O Acolher com Elas é um espaço terapêutico criado por três psicólogas, onde combinam cuidado, afeto e acolhimento.

Este projeto foi desenvolvido com o intuito de praticar o desenvolvimento web e o uso de tecnologias front-end.

## Tecnologias

- HTML
- CSS
- JavaScript
- Git e GitHub
